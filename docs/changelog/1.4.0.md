# BMPC 1.4.0 Changelog

Source range: changes after the 1.3.0 release commit `2d0329f`.

## Added

- Added a loop points editor for chamber and funnel audio. Set the loop start and end on a waveform by dragging handles or typing seconds, preview the loop, and pull loop points from the source file.
- Added correct in-game looping for exported game audio. All game assets loop by default (music and gel SFX); leaving loop points unset loops the full track.
- Redesigned the main packages window with a master–detail layout: the packages list on the left and a live detail pane on the right showing description, group, date added, song count, and a scrollable song list.
- Added a toolbar with name search (and a clear button), sorting by Name, Songs, or Date added with an ascending/descending toggle, and quick actions to refresh the list, open the packages folder, and open BEEmod.
- Added a right-click context menu on packages: New package, Edit, Remove, and Open package location (reveals the `.bee_pack` in File Explorer).
- Added keyboard and mouse shortcuts: double-click, Enter, or F2 to edit, Delete to remove, and Ctrl+N for a new package.
- Added an empty state with an icon, message, and New package button when no packages exist.
- The main window is now resizable with a sensible minimum size (previously fixed).

## Changed

- New packages and their music now use a lowercase ID scheme: package IDs are `bmpc_<name>` (was `BMPC_<NAME>_PACK`), and music IDs now include the package name for uniqueness, `bmpc_<package>_<song>_music` (was `BMPC_<SONG>_MUSIC`). This applies only to newly created or re-imported packages; existing packages keep their IDs and continue to work with no migration.
- Reduced memory use while processing package audio by working through temporary disk files instead of holding large audio data in memory.

## Fixed

- Fixed the song-count label showing incorrect text for empty packages and for the singular vs. plural case.
- Fixed the create/edit window's X button so it now shows the same cancel confirmation as the Cancel button.
- Fixed a stray cancel confirmation that appeared after a package imported successfully.

## Removed

- Removed the unused "Details" button from the packages window.

## Documentation and Release

- Release builds no longer include `.pdb` debug files.
- Updated the README.
