# BMPC 1.3.0 Changelog

Source range: changes after the 1.2.1 release commit `31c5e35`.

## Added

- BMPC as of this version is now open-source.
- Added dark mode.
- Added a theme setting with auto, light, and dark options.
- Added automatic theme matching with Windows app theme.
- Added BMPC-styled dialogs for warnings, errors, confirmations, and info messages.
- Added self-contained Windows release builds, so users no longer need to install the .NET Desktop Runtime separately.

## Changed

- Updated app windows and controls to look better in both light and dark themes.
- Updated supporting libraries to their latest stable versions.
- Updated the MaterialDesignThemes package to the latest stable version.
- Updated error and confirmation dialogs with clearer titles, icons, and button behavior.
- Updated package importing so the app stays more responsive during longer work.
- Updated copyright text in the app.
- Updated release ZIP naming to show that builds are self-contained.

## Fixed

- Fixed theme issues where some UI parts did not match the selected light or dark theme.
- Fixed dialogs opening without the correct BMPC window as their owner.
- Fixed release documentation that still described older runtime-dependent builds.
